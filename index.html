Escape Game - Le braquage

Etape 1

Dans son dernier souffle, le directeur de la banque a réussi à presser le bouton d’urgence, scellant toutes les issues et déclenchant une alarme ainsi qu’un signal lumineux rouge. Première étape pour l’équipe, désamorcer le bouton d’urgence afin d'arrêter l’alarme et accessoirement de pouvoir respirer. Malheureusement le gros bouton rouge ne semble être d’aucune utilité et pour cause: c’est une fausse piste. L'équipe doit en fait se coordonner pour appuyer sur les lumières rouges éclairant le coffre, réparties aux quatres coins de la salle, et ce, simultanément. Après avoir réussi cette épreuve, la lumière se tamise dans le coffre, l’alarme se coupe et on peut entendre de l’air circuler.

Etape 2

L'arrêt de l’alarme ne signifie pas la fin des ennuis pour l’équipe. En effet, ils sont toujours coincés derrière 50 centimètres de blindage qui ne compte pas bouger. A moins d’avoir la carte magnétique, dont le directeur s’est servi pour rentrer. La fente pour l’insérer s’illumine justement d’une lumière verte. Seul problème: la carte à disparu. Dans la panique, elle s’est mélangée à tous les billets qui jonchent le sol. L’équipe va donc devoir se lancer dans une chasse au trésor afin de trouver la carte qui s’est glissée dans une liasse de billets. Ils doivent tout de même se dépêcher, ils peuvent maintenant respirer mais la police est en chemin. Chaque seconde qui s’écoule est un pas de plus vers la prison. En répartissant la salle correctement, l’équipe devrait la trouver sans trop de soucis. Ouf, la carte est retrouvée, mais après l’avoir insérée, la porte blindée demande un code. Comment vont-ils pouvoir trouver un code caché dans ce coffre ?

Etape 3

Il n’y a pas beaucoup d'éléments dans ce coffre; des billets qui jonchent le sol et des murs recouverts de casiers. Le détail qui doit sauter aux yeux des participants, c’est que certains de ces casiers s’ouvrent et se ferment, à intervalles réguliers, 5 pour être précis. Étrange, c’est pile le nombre de caractères que requiert le code de la porte. L’équipe devra inspecter les casiers de plus près pour découvrir un code morse gravé sur chaque tiroir ouvert. Ils pourront ainsi reconstituer le code à l’aide de l'ordre d’ouverture des casiers. Si aucun des participants ne connaît le morse, le Maître du Jeu pourra ouvrir un casier supplémentaire contenant une table de correspondance morse. Le code entré est bon. Si l’équipe a réussi l’ensemble des épreuves dans le temps imparti, félicitations ! Ils s’échappent donc du coffre avec le butin. En revanche, si le temps est écoulé, ils seront accueillis à leur sortie par les policiers…

Notre super équipe voud accueille !
